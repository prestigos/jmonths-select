Web component to create a selection of months

<months-selector>

Use:

<months-selector start-year="2016" months="48"></months-selector>

Current selection:

Use `actions` property for reset/selectAll months:

<months-selector start-year="2016" months="48" actions></months-selector>

Current selection:

Atributes

selectionArray: Current selecion of Dates

Example
["2015-12","2015-11","2015-10","2015-09","2015-08","2015-07","2015-05","2015-04","2015-03","2015-02","2015-01"]

hideMonthsArray: Dates (YYYY-MM) if you dont display in selection values

Example
["2016-01", "2016-02", "2015-06"]
